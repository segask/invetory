# invetory

Система инвентаря для FiveM с привязкой к базе данных SQL.

## Установка

1. Скачайте и установите ресурс в папку resources вашего сервера FiveM.
2. Добавьте `start invetory` в server.cfg.
3. Импортируйте SQL файл `sql/inventory.sql` в вашу базу данных.
4. Убедитесь, что у вас установлен mysql-async.

## Использование

- Нажмите клавишу **I** (по умолчанию) для открытия инвентаря. Игрок может переназначить клавишу в настройках управления FiveM.
- Инвентарь сохраняется в базе данных автоматически.

## Структура

- `client/` - Клиентские скрипты
- `server/` - Серверные скрипты
- `shared/` - Общие конфигурации
- `html/` - UI интерфейс
- `sql/` - SQL схема
